API Endpoints

Users
Path Method Description
/users/signup POST Add a new user
/users/login POST Checks credentials and logs in the user.
/users GET Retrieves a list of all users.
/users/:id GET Retrieves details of a specific user by ID.
/users/:id DELETE Removes a user account by ID.
/users/:id PATCH Updates details of a specific user by ID.
Products
Path Method Description
/products POST Add a new product to the catalog.
/products GET Retrieves a list of all products.
/products/:id GET Retrieves details of a specific product by ID.
/products/:id DELETE Delete a product from the catalog by ID.
/products/:id PATCH Updates details of a specific product by ID.
/products/recent GET Retrieves the 5 most recent products.
/products/homeRecent GET Retrieves the 3 most recent products for the homepage.
